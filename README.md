## Usage

Create a config.js 

add the following boiler plate
```
module.exports = {
    MONGODB: "Your connection string here"
    TOKEN_KEY: "You secret here",
    PORT:"Your port here"
}
```
Install npm
```
npm install on both server and client
```
To start both server and client concurrently, run the following
```
npm run dev
```
